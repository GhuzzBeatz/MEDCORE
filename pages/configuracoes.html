Migracao de Dados entre PCs
Para transferir todos os dados para um novo computador, copie a pasta data localizada dentro da pasta onde o MedCore esta instalado.

Geralmente em:
C:\Program Files\MedCore\resources\app\data\
Passo a passo:
1. No PC antigo, copie a pasta data\ para um pen drive
2. No PC novo, instale o MedCore normalmente
3. Cole a pasta data\ substituindo a existente
4. Abra o MedCore e todos os dados estarao disponiveis
