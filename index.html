Acesso ao MedCore

Este acesso e simples: primeiro o app conecta ao servidor online, depois voce informa o codigo da clinica. Assim o sistema sabe em qual espaco os dados devem sincronizar.

Se voce ja usa: digite o mesmo codigo da clinica.
Se voce e novo: escolha criar uma nova clinica para gerar um codigo automaticamente.
Sincronizacao: os dados ficam salvos no computador e sobem para o servidor quando houver internet.
Seguranca: nao compartilhe o codigo da clinica fora da equipe autorizada.
Importante: cada clinica tem um codigo proprio para separar os dados.
1) Conexao online (feita uma vez)
Dica: no instalador oficial essa conexao ja vem pronta.

3) Primeiro acesso / Login
Escolha como deseja entrar:
Se voce ja trabalha em uma clinica, use o mesmo codigo da clinica que o primeiro usuario criou. Se for uma clinica nova, o sistema cria um codigo da clinica automaticamente.
4) Codigo da clinica
Use sempre o mesmo codigo para acessar os dados da clinica.
